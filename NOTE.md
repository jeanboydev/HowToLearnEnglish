## 名词

- 可数：活物

  - 单数 fish
  - 复数 
    1. fish 数量
    2. fishes：种类

- 不可数：fish 肉食


|      | 词义 | 可数性 | 三类名词   | 变形           |
| ---- | ---- | ------ | ---------- | -------------- |
|      |      | 可数   | 单数名词   | fish           |
| fish | 动物 | 可数   | 复数名词   | fish（数量）   |
|      |      | 可数   | 复数名词   | fishes（种类） |
|      | 肉食 | 不可数 | 不可数名词 | fish           |

## a, this, these

名词短语=限定词(these, two)+形容词(young)+名词

## 不可数名词

不可分隔的物质
